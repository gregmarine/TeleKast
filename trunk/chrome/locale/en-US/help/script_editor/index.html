Script Editor
The Script Editor is the starting point for everything you will do with TeleKast. This is where you will create segments for your scripts. The Teleprompter feature uses the scripts you build with the Script Editor. But that feature is not the limit for scripts. Using the Script Editor, you can create scripts to be printed and used for interviews, feature stories or just about anything you can think of. The subtopics of this Help topic will help you become familiar with the Script Editor.
