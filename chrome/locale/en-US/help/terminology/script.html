Script
Script (skript) a : something written: TEXT b : an original or principal instrument or document c (1) : MANUSCRIPT 1 (2) : the written text of a stage play, screenplay, or broadcast; specifically : the one used in production or performance.

A script is basically what TeleKast uses to communicate your message. In fact, when you open TeleKast, the Script Editor is the first thing you see. Segments, which together form the script, are added and deleted through the Script Editor. You can use the scripts for just about anything. You can print it and use it to speak in-front-of a crowd or use it in a broadcast journalism studio as a teleprompter. There is no end to what you can do with a script.
