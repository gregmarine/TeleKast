Using the Segment Clip
The Segment Clip is a convenient place to store commonly used segments. Below is a list of things regarding the Segment Clip:

Copying a segment to the Segment Clip
  • From the Script Editor menu choose Copy to Segment Clip
  • Bring up the context menu on the Script Editor (usually the right mouse button) and choose Copy to Segment Clip
  • The shortcut key Control+C
  • The toolbar button
Copying a segment to the Script
  • From the Segment Clip menu choose Copy to Script
  • Bring up the context menu on the Segment Clip (usually the right mouse button) and choose Copy to Script
  • The shortcut key Shift+C
  • The toolbar button
Deleting Segments
  • From the Segment Clip menu choose either Delete Selected Segment(s) or Delete All Segments
  • From the context menu of the Segment Clip (usually right mouse button) choose either Delete Selected Segment(s) or Delete All Segments
Refreshing the Segment Clip
  • It automatically refreshes by default (you can turn this off at any time by unchecking the Automatic Refresh check box or from the Segment Clip preferences)
  • From the Segment Clip menu, choose Refresh Segment Clip
  • From the context menu of the Segment Clip choose Refresh Segment Clip
