New/Edit Segment
The scripts you create are made of segments. Each segment has its own title, text and cues. They are the building blocks for your creative communication. This subtopic will help you get started with creating new segments.

Here is an image of the Segment Editor:


There are many ways to bring up the segment editor. Below is a list of ways to create a new segment or edit an existing one.

New Segment:
  • Go to the Script Editor menu and choose New Segment
  • Bring up the context menu on the Script (usually the right mouse button) and choose New Segment
  • Click the toolbar button
  • The shortcut key Shift+N
Edit Segment in the Script (You must have a script segment selected):
  • Go to the Script Editor menu and choose Edit Segment
  • Bring up the context menu on the Script (usually the right mouse button) and choose Edit Segment
  • Click the toolbar button
  • Double Click the segment
  • The shortcut key Control+E
Edit Segment in the Segment Clip (You must have a script segment selected):
  • Go to the Script Editor menu and choose Edit Segment
  • Bring up the context menu on the Script (usually the right mouse button) and choose Edit Segment
  • Double Click the segment
