Segment Editor
This image shows the Segment Editor:


To create a new segment in the Script, you can do any of the following: Use the shortcut key Shift-N, from the Script Editor menu select New Segment and from the context menu select New Segment.

To edit an existing segment, you can do any of the following: click on the segment you wish to edit in the Script Editor and use the shortcut key Control+E, from the Script Editor menu select Edit Segment, from the context menu select Edit Segment or double click the segment.

If you wish to edit a segment in the Segment Clip can choose Edit Segment from the Segment Clip menu, choose Edit Segment from the Segment Clip context menu or double click the segment.
  • Segment Title: Used to identify what the segment is about.
  • Segment Text: This is the body of the segment to be used by the teleprompter.
  • Camera Cue: Provides information for the camera operator.
  • Audio Cue: Provides information for the individual handling sound.
  • Video Cue: Provides information about any video to be played during the segment.
  • Talent Cue: Any information the talent needs to know for that segment.
  • Other Cue: This is for misc. information perhaps everyone should know about the segment.
When finished editing a new or previously created segment, click OK to save your work. If you do not wish to keep your new or changed segment, click Cancel, click the close button (usually an "X" in the upper right corner of the dialog) or press the Escape key.

If you are editing a new segment, you have the option to also save the segment to the Segment Clip. Simply click the "Save to the Segment Clip as well" checkbox. This saves the segment to both the Script Editor and the Segment Clip. The next time you edit a new segment, the checkbox will be check. If you do not wish to save the segment to the Segment Clip, just uncheck the box.
